PassionTextEditor

A customizable text editor with advanced features for developers and writers.

About the Project

PassionTextEditor is a dynamic, feature-rich text editor built for coding and writing tasks. This project demonstrates a strong command over HTML, CSS, and JavaScript, utilizing advanced concepts and functionalities to create a modern, intuitive, and responsive user interface.

Key Features and Concepts Used:

Project tutorial video

File Management Functionalities:

Dynamic File Handling: Created a system to handle multiple file formats (HTML, CSS, JavaScript, XML, plain text) with functionalities like New File, Open File, Save File, Save As, and Contact/Help Dialogues.

Dynamic Filename Management: Auto-fills filenames from the file content and updates tab titles dynamically.

Styling and Theming:

Built an interactive user interface with customizable themes:

Theme Color 1 & 2 allow users to choose interface colors.

Custom Text and Background Colors enhance the editor's personalization.

Reset theme options to default settings on demand.

Live Coding Preview:

Developed a Split Mode for live HTML previews with resizable screens, using DOM manipulation and event handling.

Integrated a Run HTML (Full Screen) feature to execute HTML files in real-time.

Advanced Editor Functionalities:

Implemented Auto Indentation and Line Numbers to improve code readability.

Enabled Auto Insert/Delete of Brackets and Quotes for seamless coding.

Developed Open-Close Tag Autocomplete for HTML tags, enhancing efficiency.

Custom Shorthand Support:

Incorporated shorthand techniques to generate HTML elements dynamically:

.classname → <div class="classname"></div>

tag.class#id → <tag class="class" id="id"></tag>

Supported shorthand for self-closing tags and input elements.

Custom Dialog Boxes:

Created custom-styled alert, prompt, and confirmation dialogues for features like Save, Find-Replace, Clear Text, and Filename Prompts.

Find-Replace supports text searches with line number highlights and multi-replace operations.

Tab Management System:

Designed a multi-tab interface with local storage support to restore open tabs and content upon reloading.

Integrated smooth tab scrolling and add/close functionalities.

Dynamic Interactions:

Word, Letter, and Line Count: Added a feature to calculate counts dynamically with custom alert dialogues.

Text Representation Bar: Highlights selected text and synchronizes scrolling between the editor and the representer.

UI and UX Enhancements:

Responsive and modern UI using CSS3 for animations, transitions, and smooth dropdown menus.

Included draggable and resizable custom dialogues for a polished user experience.

Additional Functionalities:

Spell Checker Toggle to enable or disable real-time spell checking.

Auto live formatting: Adds spaces and double quotes dynamically to improve code aesthetics.

Scroll Support: Smooth navigation between tabs using mouse wheel events.

Tools and Technologies Used:

HTML5: Structured the editor layout, dialog boxes, and live preview components.

CSS3: Designed a fully responsive, user-friendly interface with transitions, animations, and custom theming.

JavaScript (ES6): Implemented core functionalities including file handling, tab management, dynamic DOM manipulation, and event-driven programming.

It stands as a practical application of my knowledge in JavaScript programming, UI/UX design, and advanced web concepts.
